# WDD430 Portfolio

This repository contains my WDD430 Portfolio project built with **Next.js**, **TypeScript**, and **Tailwind CSS**. The application showcases my portfolio projects and demonstrates reusable components, file-based routing, and modern web development practices.

## Live Application

**Vercel Deployment:**

https://vercel.com/adetokefads-projects/wdd430-portfolio/BqjyPTtcWJ4b55i7yNxxkB56fst8

## Features

- Responsive portfolio website
- Reusable React components
- Next.js App Router
- TypeScript interfaces and props
- Tailwind CSS styling
- About page with a custom Skills Card component

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open http://localhost:3000 in your browser.
